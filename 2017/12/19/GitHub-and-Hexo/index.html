使用Hexo搭配Github部署個人部落格

Posted on 2017-12-19 Edited on 2020-06-27 In Hexo Disqus:

前置說明：

  • macOS
  • yarn安裝
  • Git環境設定

Hexo安裝

透過yarn安裝hexo

在初始化要建置的名稱，這邊我用”blog”

再到資料夾下進行安裝所需元件

接著在port 8080啟用hexo

直接在瀏覽器打開服務 http://localhost:8080/

基本是安裝好了

設定

Hexo 有三種預設 type: draft(草稿)、post(公開文章) 和page(頁面)，下指令時沒填 type 預設是 post。

分類

設定title與type為categories

標籤

設定title與type為tags

關於

樣式

選擇next樣式

GitHub建立repo
